一次函数解析

一次函数是形如 y = kx + b 的函数，其中：

  • k 是斜率，表示直线的倾斜程度。当k>0时，函数递增；当k<0时，函数递减；当k=0时，函数为水平直线。
  • b 是y轴截距，表示当x=0时，函数与y轴的交点。

二次函数解析

二次函数是形如 y = ax² + bx + c 的函数，图像为一条抛物线，其中：

  • a 是二次项系数，决定抛物线的开口方向与宽窄。a > 0 开口向上，a < 0 开口向下；|a| 越大，抛物线越窄。
  • b 是一次项系数，影响对称轴的位置。对称轴为 x = -b/(2a)。
  • c 是常数项，决定抛物线与 y 轴的交点 (0, c)。
  • 顶点坐标为 (-b/(2a), c - b²/(4a))，是抛物线的最高点或最低点。
  • 与 x 轴的交点由 ax² + bx + c = 0 决定：
    • 判别式 Δ = b² - 4ac > 0：两个不同的实数根
    • Δ = 0：一个二重实数根（顶点在 x 轴上）
    • Δ < 0：无实数根（抛物线不与 x 轴相交）

通过调整上方的滑块，您可以直观地看到 a、b、c 如何影响二次函数的图像、顶点、对称轴和与 x 轴的交点。
