# 📈 函数图像演示

> 动态演示一次函数 y = kx + b 与二次函数 y = ax² + bx + c 的图像，顶部可切换。拖动滑块改变参数。

## ✨ 特点

- 顶部模式切换：[一次函数] / [二次函数]
- 一次函数：滑块调节 k（斜率）和 b（截距），实时绘制直线
- 二次函数：滑块调节 a / b / c，实时绘制抛物线 + 对称轴（虚线）+ 顶点（圆点）
- 自动计算并显示关键点：顶点、对称轴、与 x 轴交点（含判别式 Δ 的三种情况）、y 轴截距
- 坐标轴自适应
- 直观理解一次 / 二次函数的性质

## 🛠️ 技术

HTML + CSS + JavaScript（Canvas 绑图）。
