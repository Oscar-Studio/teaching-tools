## 🏗️ 项目结构

```
teaching-tools/
├── index.html              # 入口页面（工具选择器）
├── tools-config.json       # 工具配置
├── user-auth.css / .js     # 用户认证样式与逻辑
├── 课堂倒计时器/
│   └── README.md           # 工具说明
├── 随机分组器/
│   └── README.md           # 工具说明
└── ...
```

每个工具均在独立文件夹中，可单独部署。
