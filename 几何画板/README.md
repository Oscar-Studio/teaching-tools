# 几何画板

> 浏览器里的交互式几何画板 — 专为几何题出图、构造与分享。

基于 SVG 渲染，零依赖（LZString 内联），支持点、线段、射线、直线、圆、多边形、角度标、文本等基础元素，以及完整的几何构造（中点、垂线、平行线、角平分线、外接圆、内切圆、四心、欧拉线…），可导出 PNG / JSON、生成可分享的 URL。

---

## 功能一览

### 基础元素
- **点** — 自动按 A, B, C, … 命名（双击可改名）
- **线段 / 射线 / 直线** — 两点定线
- **多边形** — 依次点击顶点，**再次点击第一个顶点自动闭合**（也可双击或回车闭合）
- **多边形 (下拉)** — 主按钮是自由绘制，右侧 ▼ 提供 6 种预设形状（两次点击生成）：
  - 三角形 ▶  等腰三角形（底边中点 + 顶点）/ 等边三角形（中心 + 1 个顶点）
  - 四边形 ▶  平行四边形 / 矩形 / 菱形 / 正方形（均为对角 / 相邻角 + 智能计算）
- **圆** — 圆心 + 圆上一点定圆
- **角度标** — 选 3 个点，**第 2 个为角的顶点**（更符合几何习惯）
- **文本** — 任意位置插入文字

### 几何构造
- **中点** — 两点
- **角平分线** — 角顶点和两边各一点
- **垂线** — 任意线 + 一点
- **平行线** — 任意线 + 一点
- **垂直平分线** — 两点
- **交点** — 两条线 / 线 + 圆 / 两圆（自动取第一组交点）
- **等分点** — 两点 + 输入等分数

### 三角形专属（点 3 个顶点）
- **三条高** + 3 个垂足
- **三条中线** + 3 个边中点
- **三角平分线** + 3 个交点
- **三条中位线**
- **外接圆** + 外心
- **内切圆** + 内心
- **重心 / 垂心 / 外心 / 内心** 单点

### 测量
- **长度** — 标注两点距离
- **角度** — 标注角的角度

### 文件
- **导出 PNG** — 当前画板渲染为 PNG
- **导出 JSON** — 完整可读的 JSON(pretty-printed),保留每个字段,适合存档
- **分享链接** — 用**紧凑格式 + LZString**压缩后嵌入 URL `?d=…`,打开链接即可还原
  - 紧凑格式:点坐标用 `[x,y,label?]`,其他元素用 `[type, ...点索引]`,省去所有 `id`/`a`/`b` 等冗余字段
  - 30 元素(完整三角形 + 全部构造 + 测量)分享链接约 **~400 字符**
  - 自动兼容旧版对象格式的分享链接
- **导入 JSON** — 粘贴 JSON 替换当前画板,自动识别完整格式与紧凑格式

### 视图
- **滚轮缩放** — 以鼠标位置为中心
- **拖动空白处 / 选 H 键** — 平移视图
- **G 键** — 切换栅格显示（默认隐藏）
- **⌖ 重置视图** — 回到原点

### 🧠 AI 模型解题
- 侧边栏底部 **[AI] → 🧠 模型解题** 打开右侧浮动面板
- 输入题目（如 "在△ABC中, AB=AC, ∠A=40°, 求∠B"），点 **发送**
- 模型会自动：
  1. 在画板上画点、连线、画辅助线
  2. 添加测度（角度、长度）作为推理佐证
  3. 给出完整文字解答（支持 LaTeX `$$..$$` 公式）
- 支持**多轮对话**：可在追问中继续让模型"画高"、"标角"等
- **附加画板** 选项：默认勾上，每次把当前画板状态发给模型（点坐标 + 元素），支持基于现有图继续操作
- **模型选择**：默认 `deepseek-v4-pro`（数学推理强），可在面板顶部切换，与 AI Studio 同步
- 需要登录（消耗 credits，免费模型不支持 tool_calling 故不可选）

### 通用
- **撤销 / 重做** — `Ctrl/⌘ + Z` / `Ctrl/⌘ + Y`
- **删除** — 选中后按 `Delete / Backspace`
- **Esc** — 取消当前未完成操作

## 使用流程示例：画一个三角形的所有"心"

1. 选 **点** 工具 (P)，点 3 下，画出 A, B, C
2. 选 **线段** 工具 (L)，依次连接 A→B, B→C, C→A
3. 选 **三条高** → 依次点 A, B, C — 得到 3 条高的虚线和 3 个垂足
4. 选 **重心** / **垂心** / **外心** / **内心** — 依次点 A, B, C — 4 个心位置一目了然
5. 选 **外接圆** / **内切圆** → 依次点 A, B, C — 看到两个圆
6. **分享链接** — 把 URL 发给别人，对方打开就是一样的图形
7. **导出 PNG** — 拿去做题配图

## 技术

- **渲染**：内嵌 SVG（无库），世界坐标系 y 轴向上，几何算法直接用数学向量
- **几何算法**：
  - 中点、垂足、角平分线（单位向量加法）、外心（标准公式）、内心（边长加权质心）、垂心（`H = A + B + C − 2O`）
  - 圆与直线交点（解一元二次）、两圆交点（标准公式）
- **持久化**：localStorage 自动保存当前画板
- **紧凑分享格式**：`[version, meta, points, elements]`,点用位置索引被元素引用,省去所有 `id` 字段
- **分享链接**：内联 LZString 1.5.0（MIT，~5KB），URL-safe 字符集，无需额外 urlencode
- **零运行时依赖**，完全离线可用
